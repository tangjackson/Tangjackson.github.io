Research — LLM Systems & Infrastructure

I study how to make large language models actually work at scale — covering the full stack from kernel-level GPU kernels to distributed serving clusters.

⚡

Inference Optimization

Speculative decoding, continuous batching, KV cache management (PagedAttention), and kernel fusion for low-latency serving.

🗜️

Quantization & Compression

Post-training quantization (GPTQ, AWQ), weight/activation sparsity, and mixed-precision training to shrink model footprint.

🌐

Distributed Training

Tensor, pipeline, and data parallelism; ZeRO memory optimization; gradient checkpointing across GPU clusters.

🏗️

LLM Serving Systems

Disaggregated prefill/decode, request scheduling, multi-model serving, and SLO-aware resource allocation.

Current Work

  • Efficient LLM inference — reducing TTFT and TPOT under memory constraints
  • KV cache compression and offloading strategies for long-context models
  • Disaggregated serving architectures for heterogeneous GPU clusters
